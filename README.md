# azure-cli-vm-automation
This script creates desired number of Linux VMs

Open the cloud shell and do the following steps 

1. **Create the Script File**
   
    nano deploy-linux-vms.sh
   
4. **Copy the entire code given in the file**

5. **Save and Exit**

     CTRL + O   (to Save the file)
   
     Enter
   
     CTRL + X   (to Exit the script)

7. **Make Script Executable**
   
chmod +x deploy-linux-vms.sh

9. **Run the Script**
    
./deploy-linux-vms.sh
